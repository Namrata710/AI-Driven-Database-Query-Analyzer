# AI-Driven Database Query Performance Analyzer

An AI-assisted system for analyzing database query performance and recommending indexing strategies to optimize execution time.
This project combines **Java (JDBC), MySQL, and Machine Learning** to measure query performance and intelligently suggest database indexes.

---

## Overview

Modern applications rely heavily on databases for storing and retrieving large volumes of data. Poorly optimized queries can significantly degrade system performance.

This project evaluates database query performance **with and without indexing** and integrates an **AI-based index recommendation module** to identify optimal indexing strategies.

The system executes SQL queries using JDBC, measures execution time, logs performance data, and analyzes query patterns to recommend indexes that improve performance.

---

## Features

* JDBC-based database query execution
* Query execution time measurement
* Performance comparison with and without indexing
* AI-based index recommendation system
* Machine learning model for query performance prediction
* Performance visualization dashboard
* Dataset generation for ML training
